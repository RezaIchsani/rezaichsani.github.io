Mengextract file .zip dengan Command Linux cukup mudah dengan perintah `unzip` :

- Unzip file didalam folder yang sama.

```
unzip file.zip
```

atau 

- Unzip file ke tempat lain yang berbeda.

```
unzip file.zip -d /var/www/
```
